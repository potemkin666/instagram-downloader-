🗺️ How It Works

OceanGram wraps the Osintgram toolkit in an ocean-themed web interface.

1
Clone the Repo
Fork & clone this repository, then set up your Instagram credentials.
2
Install Dependencies
Run pip install -r requirements.txt inside a virtual environment.
3
Pick a Target
Supply a public Instagram username — private accounts cannot be accessed.
4
Run Commands
Execute any OSINT command to collect followers, media, hashtags, and more.

⚡ Quick Start

Get up and running in under a minute.

# 1 — Clone
git clone https://github.com/Datalux/Osintgram.git && cd Osintgram
# 2 — Virtual environment
python3 -m venv venv && source venv/bin/activate
# 3 — Install
pip install -r requirements.txt
# 4 — Configure credentials
make setup
# 5 — Run (interactive mode)
python3 main.py <target_username>
