🗺 HOW IT WORKS

OceanGram wraps the Osintgram engine in a bioluminescent pixel-art interface.

01
CLONE THE REPO
Fork & clone, then set up your Instagram credentials in the config folder.
02
INSTALL DEPS
Run pip install -r requirements.txt inside a Python virtual environment.
03
PICK A TARGET
Supply any public Instagram username — private accounts cannot be accessed.
04
RUN COMMANDS
Use any of the 20 OSINT commands to harvest followers, media, hashtags & more.

⚡ QUICK START

Get the Osintgram backend running in under two minutes.

# 1 — Clone Osintgram
git clone https://github.com/Datalux/Osintgram.git && cd Osintgram
# 2 — Create & activate virtual env
python3 -m venv venv && source venv/bin/activate
# 3 — Install dependencies
pip install -r requirements.txt
# 4 — Set up Instagram credentials
make setup
# 5 — Launch interactive shell
python3 main.py <target_username>
